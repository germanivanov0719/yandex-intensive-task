Проекты

Сайт для расчета среднего балла и количества оценок до целей. Идея для этого сайта появилась, когда с обновлением электронного дневника в Москве он начал неправильно считать средний балл, и мои одноклассники не могли понять, сколько им нужно получить оценок до какого-то результата в триместре. Сайт позволяет настраивать разные параметры (например, дробную часть для округления вверх) и не ограничен подсчетами только в верхнюю сторону (как электронный дневник).

С точки зрения технологий этот проект интересен тем, что он сделан в виде PWA с кешированием, что позволяет ему работать офлайн (пока браузер хранит кеш). Пользовательские настройки сохраняются в localStorage. Сайт не требует настройки серверной части, так как все вычисления проходят локально (код написан на JavaScript). Клиентская часть написана на обычном HTML с Bootstrap и небольшим количеством моих стилей.

Ссылки: сайт, репозиторий GitHub. Исходный код доступен под MIT License.

Сайт не с самой оригинальной идеей, но достаточно интересной серверной частью. Есть REST API, используются Flask с Jinja, данные хранятся в SQLite. Клиентская часть также реализована с помощью Bootstrap.
